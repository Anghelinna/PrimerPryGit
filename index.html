REPOSITORIO GIT & GITHUB LA RAMA PRINCIPAL

Comandos:

  • git init
  • git add .
  • git commit -m "Mensaje de commit"
  • git commit -am "Agregado el add y Mensaje de commit"
  • git commit --amend -m "Sirve para editar el mensaje del último commit realizado (Editor Vim)"
  • git status -s: Verificar el estado del repositorio
  • git log --oneline: Ver lista de todas las versiones
  • git reset --hard (codigo alfanuméreico): Volver a una versión anterior
  • git config --global user.name "Nombre de usuario"
  • git config --global user.email "Correo electrónico"
  • git config --list: Visualizar todas las configuraciones
  • git config user.name: Verificar el nombre de usuario configurado
  • git config user.email: Verificar el correo electrónico configurado
